Instructions

  1. If this is your first visit, go ahead and click the register button in the top right corner and sign up for an account.
  2. After you register, you can log in.
  3. To start playing a game, go to the "My Battles" page. This page shows all the battles you have participated in and allows you to start a new battle.
  4. You can start a new battle by entering another player's name or by clicking the "Find an Online Player" button. If you ask the game to find you a match you will be notified when a match is found by a popup on the "My Battles" page.
